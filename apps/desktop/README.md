# Desktop App

Desktop app for Hubble.md (TypeScript + Tauri v2).

## Run development server

From repo root:

```sh
pnpm --filter @hubble.md/desktop dev
```

Or from `apps/desktop`:

```sh
pnpm dev
```

## Build for production

From repo root:

```sh
pnpm --filter @hubble.md/desktop build
```

Or from `apps/desktop`:

```sh
pnpm build
```
